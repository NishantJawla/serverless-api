# serverless-api

## Technologies

-   Node.js + Express.js
-   DynamoDB
-   Serverless

## Instructions to run

-   Pre-requisites:

    -   Node.js
    -   NPM

-   Directions to setup/install

```bash
$ git clone (copy the http link from clone button)
$ cd (filenamecreated)
$ touch .env
$ cp .env.example .env
# Set values in .env
$ npm i
```

-   Directions to execute

```bash
$ npm run dev
```
